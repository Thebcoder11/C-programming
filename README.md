## 📖 About This Repository
This repository is a comprehensive log of my journey in learning and mastering **C Programming**. It acts as both a personal archive and an educational resource. 

C is considered the mother of all modern programming languages. Whether you want to understand memory management, build operating systems, or just learn how computers work under the hood, C is the perfect starting point.

Explore the repository by navigating through the neatly structured topics. Every folder contains source code and specific examples to learn from.

| 📁 Topic | 📝 Description |
| :--- | :--- |
| **[🌱 Basic](./Basic)** | Understand the structure of a C program, `#include`, `printf()`, `scanf()`, variables, and primitive data types. |
| **[🔀 Conditional Statement](./Conditional%20Statement)** | Learn to make decisions using `if`, `else if`, `else`, nested conditions, and the `switch-case` statement. |

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing.

### 1️⃣ Prerequisites
To compile and run the code, you need a C Compiler installed.
*   **Windows:** Install [MinGW-w64](https://www.mingw-w64.org/) or [MSYS2](https://www.msys2.org/).
*   **macOS:** Open Terminal and type `xcode-select --install`.
*   **Linux:** Open Terminal and type `sudo apt install build-essential`.

### 2️⃣ Clone the Repository
```bash
git clone https://github.com/your-username/C-Programming.git
cd C_Programming
```

### 3️⃣ Compile and Run

Navigate to the directory of the topic you want to test. For example:
```bash
cd "Function And Pointer"
```

**Compile using GCC:**
```bash
gcc program_name.c -o my_program
```

**Run the generated executable:**
*   **Windows:**
    ```cmd
    .\my_program.exe
    ```
*   **Linux/macOS:**
    ```bash
    ./my_program
    ```
